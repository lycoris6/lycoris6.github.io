# 命理分析网站交互设计

## 核心交互功能

### 1. 名字数据筛选器
- **五行筛选**: 用户可以根据五行属性（金、木、水、火、土）筛选名字
- **生肖适配**: 根据生肖蛇的宜用字根进行筛选
- **吉凶等级**: 按五格数理的吉凶等级筛选（吉、半吉、凶）
- **笔画范围**: 根据名字总笔画数进行范围筛选

### 2. 智能搜索系统
- **名字搜索**: 支持拼音和汉字搜索
- **寓意搜索**: 根据寓意关键词搜索（如"平安"、"才华"、"稳重"等）
- **卦象搜索**: 根据周易卦象名称搜索
- **综合评分**: 根据综合适配度排序

### 3. 名字对比工具
- **多选对比**: 用户可以选择2-4个名字进行详细对比
- **雷达图展示**: 使用雷达图展示各名字在不同维度的评分
- **优劣分析**: 自动生成的对比分析报告

### 4. 详细分析报告
- **点击展开**: 点击任意名字卡片展开详细分析
- **分页展示**: 详细报告分为基本信息、五行分析、数理分析、卦象解读等分页
- **风险提醒**: 突出显示需要注意的风险点

### 5. Excel导出功能
- **自定义导出**: 用户可以选择需要导出的字段
- **批量导出**: 支持批量导出筛选后的所有名字数据
- **格式选择**: 提供多种Excel格式模板选择

## 用户交互流程

1. **进入网站**: 用户看到所有名字的整体概览
2. **筛选数据**: 使用左侧筛选面板缩小选择范围
3. **查看详情**: 点击感兴趣的名字查看详细分析
4. **对比选择**: 选择多个名字进行对比分析
5. **导出数据**: 将选中的名字数据导出为Excel文件

## 响应式设计

- **桌面端**: 三栏布局（筛选器 + 名字列表 + 详情面板）
- **平板端**: 两栏布局（筛选器 + 名字列表，详情覆盖显示）
- **移动端**: 单栏布局（筛选器可折叠，详情全屏显示）
